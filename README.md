# NatureScapes

Cloud-native multimedia platform for uploading and managing nature photography and videography.

## Live URL

Frontend:
https://cloudnativeb924368.z43.web.core.windows.net/

---

## Architecture

```text
Browser Frontend
        ↓
Azure Function API
        ↓
Azure Blob Storage (media files)
Azure Cosmos DB (metadata)
```

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Azure Functions
- Azure Blob Storage
- Azure Cosmos DB

---

## Features

- Upload images and videos
- Cloud-based storage
- Responsive UI
- Dynamic gallery
- REST API backend
